# JobPortal

JobPortal is a simple ASP.NET Core MVC web application built to practise C# and the MVC pattern. The project allows users to enter the system either as an admin or as an applicant and demonstrates basic form handling, routing, and page redirection.

## Features
- Role-based redirection (Admin, Applicant and Organizations)
- ASP.NET Core MVC architecture
- Razor views with HTML and Bootstrap for layout
- Simple form input and controller logic

## Technologies Used
- C#
- ASP.NET Core MVC
- HTML
- Bootstrap

## How to Run
1. Open the project in Visual Studio.
2. Restore NuGet packages if needed.
3. Run the project using IIS Express or `dotnet run`.
4. Open the browser at the displayed localhost address.

## Purpose
This project is created for learning and academic practice. It focuses on understanding MVC flow.
